# 🚧 Intelli Road - Smart Road Monitoring System

## 📌 Overview
Intelli Road is an AI and IoT-based system designed to monitor road conditions in real-time.  
It detects potholes, road damage, and unsafe driving behaviors using Computer Vision and IoT sensors.

---

## 🚀 Features
- Real-time pothole detection using Computer Vision
- Road damage identification
- Unsafe driving detection
- IoT-based data collection
- Live monitoring dashboard (concept)

---

## 🛠️ Tech Stack
- Python
- YOLOv8
- Computer Vision
- Raspberry Pi
- IoT Sensors

---

## ⚙️ How It Works
1. Camera captures real-time road footage
2. YOLOv8 model detects potholes and damages
3. IoT sensors collect environmental data
4. Data is sent to a monitoring dashboard
5. Authorities can take action based on insights
